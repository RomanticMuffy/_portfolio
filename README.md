## Sobre

Portfólio pessoal de **Gustavo Henrique Cobello Lima** — desenvolvedor full-stack e designer UI/UX. O site funciona como um cartão de visita digital, apresentando minhas habilidades, projetos em destaque (buscados dinamicamente via GitHub API) e formas de contato.

Construído inteiramente com **HTML**, **CSS** e **JavaScript** puros — sem frameworks, sem build tools, sem dependências. Tudo nasce do zero, do design ao deploy.

## Sobre Este Projeto

Este portfólio foi pensado para ser mais do que uma lista de tecnologias — é uma experiência visual que reflete minha abordagem como profissional híbrido de design e desenvolvimento. Cada seção foi desenhada para contar uma parte da minha história:

- **Hero** — apresentação direta com ilustração autoral
- **Sobre Mim** — bio profissional dividida em parágrafos e competências organizadas por domínio (Design & UI/UX, Front-End, Back-End, DevOps & Infra, Segurança & Redes)
- **Projetos** — meus 3 repositórios mais recentes buscados ao vivo da API do GitHub, com cards decorativos
- **Links Sociais** — GitHub, LinkedIn, e-mail

---

## Funcionalidades

- **Zero dependências** — HTML, CSS e JavaScript puros, nada para instalar
- **Tema claro/escuro** — alternância com um clique, invertendo cores e cursores personalizados
- **Animações customizadas** — transições CSS escritas à mão e interações JS, sem bibliotecas de animação
- **Cursores personalizados** — SVGs customizados para tema claro e escuro (padrão + ponteiro)
- **Integração com GitHub API** — busca e exibição dinâmica de repositórios recentes
- **Design responsivo** — adapta-se entre telas desktop e mobile (breakpoint em 768px)

## Design Visual

A linguagem visual do site é intencional e baseada em pesquisa. O design não começou no código — começou no **Figma**, passando por um processo completo: de wireframes de baixa fidelidade a um sistema visual de alta fidelidade, antes de uma única linha de HTML ser escrita.
